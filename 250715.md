* Java 데이터 타입

1. 기본형 : 고정된 크기와 범위를 가진다.

|정수형|크기 (byte)|표현 범위|
|------|---|---|
|byte|1|-128 ~ 127|
|short|2|-32,768 ~ 32,767|
|int|4|-2³¹ ~ 2³¹ - 1<br/>(-2,147,483,648 ~ 2,147,483,647 ➡️ 약 21억)|
|long|8|-2⁶³ ~ 2⁶³ - 1|


|실수형|크기 (byte)|
|------|---|
|float|4|
|double|8|

|문자형|크기 (byte)|
|------|---|
|char|2|

|논리형|크기 (byte)|
|------|---|
|boolean|1|


2. 참조형 : 표현 범위가 고정되지 않음. 논리적으로 표현 범위는 무한에 가깝지만, ~실제론 시스템 자원에 제한됨~

##

💁‍♀️ 자바는 기본형 변수에 실제 값을 저장하고, 참조형 변수에는 객체의 메모리 주소를 저장함

💁‍♀️ double이 float보다 더 넓은 범위와 정밀도를 지원하므로 더 많이 사용됨

```java
int a = 1000000000; // 최대 ±2,147,483,647까지만 표현 가능 (고정)

String s = "Hello world"; 
// 문자열 길이에 제한 없음. 메모리만 허용한다면 수억 자도 가능
```
